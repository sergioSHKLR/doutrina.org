# doutrina-content

Conteúdo em Markdown puro dos livros de Allan Kardec para doutrina.org.

**Como contribuir**
- Edite apenas os arquivos .md (nunca adicione HTML/CSS).
- Mantenha a mesma estrutura de headings (#, ##, ###).
- Use classes existentes (ex: {:.grey-center}, {:.spirit-multi}).
- Após editar, abra um Pull Request ou avise no doutrina.org repo.

Para gerar o site: copie a pasta `books/` para o repositório doutrina.org (ou use git submodule).
